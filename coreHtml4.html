Some HTML Tags and their description

Basic tags
  1. Head tag :-

    2. The head tag < head > contains all the elements describing the document.

  2. Title tag :-

    3. The title tag < title > specifies the HTML page title, which is shown in the browser’s title bar.

  3. Body tag :-

    4. The body tag < body > is where you insert your web page’s content.

  4. Paragraph tag :-

    5. A paragraph tag < p > is used to define a paragraph on a web page.

  5. Heading tag :-

    6. The HTML heading tag is used to define the heading of the HTML document. The < h1 > tag defines the most important tag, and < h6 > defines the least.


Formatting tags
  1. Emphasis tag :-

    2. The HTML < em > tag is used to emphasize the particular text in a paragraph.

  2. Bold tag :-

    3. The < b > tag is used to make the text bold.

  3. Italic tag :-

    4. The < i > tag is used to make the text italic.

  4. Underline tags :-

    5. The < u > tag is used to set the text underline.

  2. Image tag :-

    3. The < img > tag is used to embed an image in an HTML document. You need to specify the source of the image inside the tag.

  3. Table tag :-

    4. The < table > tag is used to create a table in the HTML document.
    The table row (< tr >) tag is used to make the rows in the table, and the table data (< td >) tag is used to enter the data in the table.

  4. Strong tag

    5. The < strong > tag in HTML is the parsed tag and is used to show the importance of the text. Make that text bold.

  5. Paragraph tag

    6. The < p > tag in HTML defines a paragraph. These have both opening and closing tags.

  6. Object

    7. The object tag is an HTML tag used to display multimedia like audio, videos, images, PDFs, and Flash on web pages.
